# 簡易的なチャットシステム

## 概要

クライアントアプリととサーバーアプリがある
サーバーはテキストを管理して、データを保存している。
クライアントはサーバーにリクエストを送ることでデータを得ることができる。

## 使い方

### サーバー

#### 起動方法

```sh
python host.py
```

### クライアント

#### 起動方法

```sh
python client.py
```

#### 送受信

モード0 メッセージ送信  
入力 送信するテキスト  
出力 サーバ上でのテキストのID  
モード1 メッセージ受信  
入力 受信したいテキストのID  
出力 テキスト  

一度受け取りリクエストしたデータは消える
